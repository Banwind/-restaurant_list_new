my favorite restaurant

##Project function

A little tool to recording your own favorite restaurants and make your own pocket list.

##Features

-add manage page for batch update and modify.

輸入以下內容，安裝與本專案相關的套件：

```
npm install

npm run seed

npm run dev
```

進入網址

```
http://localhost:3000/
```

想結束請按 ctrl + c

# 環境建置與需求：

- express
- express-handlebars
